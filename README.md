# Component library

A component library following [Atomic Design](http://bradfrost.com/blog/post/atomic-web-design/) principle and Storybook

## Installation

```bash
yarn
```

To start storybook:

```bash
yarn run storybook
```

To start project:

```bash
yarn start
```
